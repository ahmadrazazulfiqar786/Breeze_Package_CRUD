# 📦 Breeze Package CRUD API

## 📖 Project Overview

**Breeze Package CRUD** is a Laravel-based **API-only** application that demonstrates how to implement authentication and CRUD operations using **Laravel Breeze (API stack)** with **Laravel Sanctum** for token-based authentication. The project also integrates **Swagger (L5-Swagger)** to provide interactive API documentation and testing.

This project does **not include any frontend or Blade views**. All interactions are handled through RESTful APIs.

---

## 🎯 Objectives of the Project

* ✅ Implement API authentication using **Laravel Breeze**.
* ✅ Use **Laravel Sanctum** for secure token-based authentication.
* ✅ Develop **CRUD (Create, Read, Update, Delete)** operations for a Product resource.
* ✅ Integrate **Swagger (OpenAPI)** for interactive API documentation.
* ✅ Apply **Form Request Validation** for clean and secure input handling.
* ✅ Structure the project following Laravel best practices.
* ✅ Provide a ready-to-use backend for mobile or frontend applications.
